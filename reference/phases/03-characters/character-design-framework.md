# 角色设计框架

> 综合参考：Lajos Egri 《The Art of Dramatic Writing》三维角色理论、Robert McKee 《Story》、K.M. Weiland 角色弧线方法论。融入 [novel-writer-skills](https://github.com/wordflowlab/novel-writer-skills) 的"角色弧线 + 关系追踪"系统、[ClawHub novel-writing](https://clawhub.ai/zton100/novel-writing) 的"角色声线卡"机制。

## 一、角色卡模板（主角版）

```markdown
# [角色名] — 角色档案

## 基础信息
- 姓名/称号：
- 年龄：
- 外貌特征（3个记忆点）：
- 职业/身份：
- 出生背景：
- 教育/经历：

## 三维角色（Lajos Egri 理论）
### 生理维度（Physiological）
- 性别、年龄、身高、体重
- 健康状况、外貌、体态
- 缺陷/疤痕/特征

### 社会维度（Sociological）
- 阶层、收入、职业
- 家庭、教育、宗教
- 政治倾向、爱好、嗜好

### 心理维度（Psychological）
- 性格、气质、世界观
- 雄心、挫折、欲望
- 恐惧、复杂、互补
- IQ / EQ

## 性格内核（必填）
- 三词定义法：[词1] + [词2] + [词3]
  （示例：聪明+社恐+嘴硬心软）
- 核心欲望：最想得到什么？
- 核心恐惧：最害怕什么？
- 致命缺陷（Fatal Flaw）：什么弱点会害了他？
- 谎言信念（The Lie）：他相信的错误认知是什么？
- 真相（The Truth）：他需要学到的真理是什么？
- 伪自我（False Self）：他对外展示的样子
- 真自我（True Self）：他内心真正的样子

## 角色弧线
- 起点状态：故事开始时是什么样？
- 催化改变：什么事件开始推动改变？
- 中点觉醒：意识到了什么？
- 最低谷：失去了什么/面对了什么？
- 终点状态：故事结束时变成什么样？

## 行为模式
- 压力下的反应：
- 说话方式/口癖：
- 习惯动作/小动作：
- 对陌生人的态度：
- 对亲近人的态度：
- 应对失败的方式：
- 应对成功的方式：

## 声线卡（Character Voice Card）
- 高频词/口头禅：[3-5个]
- 句式偏好：（长句/短句/反问/陈述）
- 比喻习惯：（科学/文学/日常/俗语）
- 情绪表达：（外放/内敛/讽刺/温柔）
- 禁忌词：（绝不会说的话/词）

## 关系网络
| 角色 | 关系 | 互动模式 | 冲突点 | 演变方向 |
|------|------|---------|--------|---------|
| | | | | |

## 秘密
- 不为人知的过去：
- 当前隐瞒的事：
- 自己也不愿承认的：

## 触发器（Triggers）
- 让他失控的事：
- 让他柔软的事：
- 让他狠心的事：
```

## 二、角色弧线设计方法（K.M. Weiland）

### 核心公式：The Lie + Wound + Want + Need

```
Wound（伤痛）→ 形成 Lie（谎言信念）
                    ↓
The Lie → 角色 → Want（欲望，表层目标）
                    ↓
                Need（需求，深层真理）
                    ↓
                故事结束：是否突破 Lie 接受 Need？
```

### 三种核心弧线

#### 1. 正向弧线（Positive Arc）— 最常见
```
相信谎言 → 受到挑战 → 抵抗改变 → 被迫面对 → 接受真相 → 改变行为
```
- **起点**：相信 The Lie
- **催化**：事件挑战 The Lie
- **抵抗**：用 Lie 应对失败
- **顿悟**：看到 The Truth
- **改变**：接受 Need，放下 Want

**示例**：
- 自卑的人学会相信自己（Lie: "我不配" → Truth: "我值得"）
- 复仇者放下仇恨（Lie: "复仇能让我解脱" → Truth: "宽恕才是解脱"）

#### 2. 负向弧线（Negative Arc）— 悲剧/反派
**A. 堕落型**：好人走向黑暗
```
相信真相 → 受到诱惑 → 接受谎言 → 抛弃真相 → 堕落
```

**B. 加倍型**：执念越来越深
```
相信谎言 → 受到挑战 → 拒绝改变 → 加倍投入谎言 → 毁灭
```

**C. 幻灭型**：真相过于残酷
```
相信谎言 → 看到真相 → 拒绝接受 → 自我崩塌 → 绝望
```

**示例**：好人因为执念变成反派、英雄成为他曾经讨厌的样子

#### 3. 平坦弧线（Flat Arc）— 传奇英雄型
```
知道真相 → 进入不信真相的世界 → 通过行动改变周围人 → 世界接受真相
```
- 主角不变，世界变
- 主角是"真理的载体"
- 适合系列作品的核心英雄

**示例**：阿甘、绿巨人浩克、佐罗、捍卫战士

### 弧线检验清单

- [ ] The Lie 在第1幕清晰展示（通过行为不通过台词）
- [ ] Wound 在故事中至少暗示一次（不必明说）
- [ ] Want 是表层目标，与 Need 形成张力
- [ ] 中点：主角第一次"瞥见"真相
- [ ] 暗夜灵魂：主角最深的怀疑/恐惧
- [ ] 第3幕：主角主动选择放下 Lie
- [ ] 结局：行为证明 Truth 已被接受（不通过台词，通过行动）

## 三、角色辨识度设计

### 对话辨识度检测

**标准**：遮住角色名，只看对话，能不能辨认出是谁在说话？

| 维度 | 设计方法 |
|------|---------|
| 用词习惯 | 每个角色有 2-3 个专属高频词/口癖 |
| 句式结构 | 有人爱用长句分析，有人说话断断续续 |
| 信息密度 | 有人废话多，有人惜字如金 |
| 潜台词程度 | 有人直接说，有人永远绕圈 |
| 情绪表达 | 有人外放，有人压抑 |
| 文化层次 | 用词水平区分（书面/口语/俚语/方言） |
| 礼貌度 | 称呼习惯（你/您/小+名字/直呼其名） |

### 声线卡示例

**A 角色**：理性程序员，社恐
- 高频词：理论上、概率、按道理
- 句式：分点陈述、长句包含定语从句
- 比喻：技术类比（"像 bug 一样"、"内存溢出"）
- 禁忌：不说"我爱你"，会说"我系统已为你保留专属端口"

**B 角色**：暴脾气退伍兵
- 高频词：他妈的、爷、操、滚
- 句式：超短句、命令式、感叹句多
- 比喻：军事/暴力类比（"像挨了一枪"、"火力压制"）
- 禁忌：从不解释自己，从不示弱

**C 角色**：温婉江南女子
- 高频词：嗯、罢了、挺好的
- 句式：完整长句但语气柔和、多用问句
- 比喻：自然/古典类比（"像春雨"、"像旧瓷器"）
- 禁忌：从不直接说"不"，会用迂回方式

### 外在标记设计

每个重要角色至少有 **1 个视觉标记 + 1 个行为标记**：
- **视觉标记**：疤痕、发色、穿着风格、标志性配饰、身高体态
- **行为标记**：习惯动作、独特姿势、重复行为、口头禅

### 单维度禁令
**禁止单一标签角色**：
- ❌ 仅"高冷霸道总裁"
- ✅ 高冷霸道总裁 + 偷偷写诗 + 怕老鼠 + 收藏奇怪手办

## 四、配角设计原则

### 配角的三个功能

1. **镜子功能** — 映照主角的某个侧面（正面或反面）
2. **催化功能** — 推动/阻碍主角的改变
3. **世界功能** — 展示世界观的某个切面

### 配角复杂度层级

| 层级 | 复杂度 | 适用 | 设计深度 |
|------|--------|------|---------|
| L1 功能性 | 低 | 路人/工具人 | 1-2 个特征即可 |
| L2 有趣性 | 中 | 常见配角 | 有自己的小目标和小矛盾 |
| L3 完整性 | 高 | 重要配角 | 完整动机+自己的弧线 |
| L4 主角级 | 极高 | 多 POV 中的其他主角 | 等同主角设计 |

### 配角设计的"3要1不要"原则

**3要**：
1. **要有自己的目标**（哪怕和主线无关）
2. **要有自己的弱点**（让读者共情）
3. **要有自己的"在场理由"**（不是为主角服务）

**1不要**：
- 不要为了"金句""冷场救场""信息传递"而存在 → 工具人感

### 4 大配角原型

| 原型 | 功能 | 示例 |
|------|------|------|
| 导师（Mentor） | 引导主角成长 | 邓布利多、奎刚 |
| 阴影（Shadow） | 主角的反面 | 反派/对手/暗黑面 |
| 守门人（Threshold Guardian） | 阻止主角进入新世界 | 第一个 Boss、严父 |
| 变形者（Shapeshifter） | 立场不明确，制造怀疑 | 双面间谍、亦正亦邪 |

## 五、反派设计

### 好反派的标准

- ✅ 他认为自己是对的（从他的视角看逻辑自洽）
- ✅ 他有可以被理解（甚至同情）的动机
- ✅ 他的能力足以构成真正威胁
- ✅ 他和主角在某个层面是"镜像"
- ✅ 他在某个时刻让读者动摇（"也许他有道理？"）

### 反派动机光谱

| 类型 | 动机 | 示例 |
|------|------|------|
| 理想型 | 为"更好的世界"不择手段 | 灭霸（平衡宇宙） |
| 创伤型 | 过去的伤害导致极端行为 | 大多数复仇型反派 |
| 贪欲型 | 对权力/金钱/控制的渴望 | 经典权谋反派 |
| 哲学型 | 与主角代表不同价值观 | 小丑 vs 蝙蝠侠 |
| 镜像型 | "如果主角走了另一条路" | 主角的暗面映射 |
| 系统型 | 反派是体制本身，没有具体人 | 《1984》大哥、卡夫卡式 |
| 自然型 | 自然灾害/疾病/末日 | 《老人与海》的鲨鱼、瘟疫 |

### 反派与主角的镜像关系

最有力的反派与主角共享同一个 Wound，但选择不同：

| 主角 | 反派 |
|------|------|
| 同样的童年伤痛 | 同样的童年伤痛 |
| 选择正面应对 | 选择拥抱黑暗 |
| The Truth | The Lie |

**示例**：
- 蝙蝠侠 vs 小丑：都因创伤变得不正常，蝙蝠侠选择"秩序"，小丑选择"混乱"
- 哈利波特 vs 伏地魔：都是孤儿+混血+蛇佬腔，哈利选择爱，伏地魔选择恐惧

### 反派检测题

- 反派从他自己的视角看，他的行为合理吗？
- 反派有没有让读者短暂同情/理解的瞬间？
- 把反派的故事单独写一本书，能不能成立？
- 反派和主角能不能"互换"？（如果不能，可能层次差太多）

## 六、关系图谱设计

### 核心关系类型

| 关系 | 叙事功能 | 冲突来源 |
|------|---------|---------|
| 师徒 | 成长线 + 传承主题 | 超越/背叛/失去 |
| 对手 | 压力源 + 成长催化 | 竞争/理念冲突 |
| 搭档 | 互补 + 日常趣味 | 分歧/信任危机 |
| 感情 | 情感线 + 软肋 | 误解/选择/牺牲 |
| 仇敌 | 外部压力 + 过去纠葛 | 复仇/原谅 |
| 亲情 | 根源 + 内心矛盾 | 期望/反叛/和解 |
| 友情 | 支持 + 底色 | 价值观/路线分歧 |

### 关系动态化原则

- 关系不是静态的 — 每个重大事件后关系应该有变化
- 用"秘密"驱动关系变化 — A 知道 B 不知道的事 → 张力
- 三角关系制造冲突 — 不只是感情三角，任何三人利益冲突都有效

### 关系矩阵（多角色作品必备）

| | 角色A | 角色B | 角色C | 角色D |
|---|---|---|---|---|
| 角色A | — | 兄弟/嫉妒 | 师父/敬畏 | 仇人/仇恨 |
| 角色B | 兄弟/嫉妒 | — | 暧昧/未明 | 陌生 |
| 角色C | 师父/慈爱 | 暧昧/克制 | — | 敌对/警惕 |
| 角色D | 仇人/利用 | 不认识 | 敌对/想杀 | — |

**注意**：A 对 B 和 B 对 A **不一定相同**（例如：A 嫉妒 B，但 B 视 A 为亲兄弟）

### 关系演变追踪表

每章末记录：

```markdown
## 第N章 关系变化

| 角色对 | 之前 | 现在 | 触发事件 |
|--------|------|------|---------|
| A→B | 兄弟 | 仇人 | 第N章 B 杀了 A 的师父 |
| C→D | 警惕 | 敌对 | 第N章 D 暴露身份 |
```

## 七、群像作品的角色管理

### 角色权重分层

| 层级 | 数量 | 笔墨占比 | 设计深度 |
|------|------|---------|---------|
| 主角（POV） | 1-3 | 60-80% | 完整三维 + 弧线 |
| 重要配角 | 5-10 | 15-30% | 完整动机 + 自己的小弧线 |
| 关键路人 | 10-30 | 5-10% | 1-2 个特征 + 功能 |
| 背景角色 | 不限 | <5% | 仅名字 + 一句话 |

### POV（视角）切换规则

- 一个场景一个 POV — 不要在一个场景内换视角
- 每章固定一个 POV — 给读者明确的"代入对象"
- POV 角色必须**有信息差** — 不同 POV 知道不同的事
- POV 切换要有**节奏感** — 不要每章都换，也不要 5 章不换

### 群像作品的"角色出场顺序"

第 1-3 章只能出现 ≤5 个角色（读者大脑容量有限）

之后每 2-3 章引入 1 个新重要角色（带着记忆点出场）

## 八、角色一致性检查（防"OOC"）

### OOC（Out Of Character）的定义

角色做出违背其人设的行为，没有合理铺垫。

### 一致性检查清单

- [ ] 这个角色在压力下的反应，符合他的"压力反应"设定吗？
- [ ] 这段对话的用词，符合他的"声线卡"吗？
- [ ] 他做这个决定的时候，有没有内心挣扎？（没有 = 太顺=不真实）
- [ ] 这个行为与他的核心欲望/恐惧一致吗？
- [ ] 他知道这件事吗？（信息一致性，避免全知）
- [ ] 他现在的能力等级与之前一致吗？（修真/玄幻题材）

### 角色"突变"的合法路径

如果角色行为发生重大变化，必须经过：

1. **触发事件**（重大刺激）
2. **挣扎过程**（内心冲突外化）
3. **关键选择**（有意识的决定）
4. **行为变化**（之后所有行为体现新状态）

❌ 突然变化（无铺垫）= OOC = 灾难
