# 阶段 03：人物工坊

## 触发条件

用户说"设计角色""新人物""这个角色怎么"时加载本手册。

## 流程

### 1. 定位角色

先不急着填卡。确认角色在故事中的位置：

- **角色层级**：L1 功能性 / L2 有趣性 / L3 完整性 / L4 主角级？
- **对主角的功能**：镜子（映照主角）/ 催化（推动弧线）/ 世界（展示世界观）/ 对手（提供冲突）？
- **在核心冲突中的位置**：推动者 / 阻碍者 / 变量 / 代价？
- **首次登场章**

### 2. 性格内核

**Wound → Lie → Want ↔ Need**（K.M. Weiland 弧线公式）：

1. **Wound（伤痛）**：过去发生了什么创伤？
2. **Lie（谎言信念）**：Wound 让他相信了什么错误认知？
3. **Want（核心欲望）**：Lie 驱动下想要什么？— 角色自己知道
4. **Need（深层需求）**：真正需要什么？— 角色不知道，故事要教他

例：莉莉丝 — Wound=目睹母后暴毙 → Lie="我不配当王" → Want=证明自己 → Need=王不在完美而在真心

**三词定义法**：[形容词] + [形容词] + [形容词]，三个词概括性格核心。

**致命缺陷**：什么弱点会害了他？— 必须是一个会在故事中造成实际后果的弱点，不是小缺点。

**双面自我**：对外展示的样子 vs 内心真正的样子。差距越大 = 戏越足。

### 3. 弧线设计

选择弧线类型 + 填写 5 个节点：

| 类型 | 轨迹 |
|------|------|
| **正向弧线** | 信 Lie → 被挑战 → 抵抗 → 被迫面对 → 接受 Truth → 改变 |
| **负向·堕落** | 信 Truth → 受诱惑 → 接受 Lie → 抛弃 Truth → 堕落 |
| **负向·加倍** | 信 Lie → 被挑战 → 拒绝改变 → 加倍 Lie → 毁灭 |
| **负向·幻灭** | 信 Lie → 看到 Truth → 拒绝接受 → 崩塌 → 绝望 |
| **平坦弧线** | 已知 Truth → 改变周围人 → 世界接受 Truth（主角不变） |

弧线检验 7 问：
- [ ] Lie 在故事中通过行为（不只台词）展示了？
- [ ] Wound 至少暗示过一次？
- [ ] Want 与 Need 形成张力？
- [ ] 中点：主角第一次瞥见 Truth？
- [ ] 暗夜灵魂：有最深的怀疑/恐惧时刻？
- [ ] 第 3 幕：主动选择放 Lie？
- [ ] 结局：行动证明 Truth 被接受？（不靠台词，靠行为）

### 4. 声线卡

**标准：遮住名字，只看对白，能认出谁在说话。**

至少填 5 项：
- 高频词/口头禅（3-5 个）
- 句式偏好（长句/短句/反问/陈述）
- 比喻习惯（科学类比/文学引用/日常俗语）
- 禁忌词（绝不会说的话）
- 用词水平区分（书面/口语/俚语/方言）
- 称呼习惯（你/您/名字/绰号）
- 情绪表达（外放/内敛/讽刺/温柔）
- 信息密度（废话多/惜字如金）
- 潜台词程度（直接说/永远绕圈）

### 5. 外在标记

至少 1 个视觉标记 + 1 个行为标记：
- 视觉：疤痕/发色/穿着风格/配饰/体态
- 行为：习惯动作/小动作/重复行为

**单维度禁令**：禁止"仅高冷""仅活泼"。至少叠加两个矛盾特质。

### 6. 反派专项

仅当角色是主要反派时执行。

**好反派标准**：
- 他认为自己是对的（从他的视角逻辑自洽）
- 动机可被理解（不一定要被认同）
- 能力构成真正威胁
- 与主角在某个层面是"镜像"

**动机类型**：理想型 / 创伤型 / 贪欲型 / 哲学型 / 镜像型

**镜像关系**：最强反派与主角共享同一个 Wound，但选择了不同的路——主角选了 Truth，反派选了 Lie。

### 7. 关系网

与已有角色的关系。标注：关系类型 + 温度（1-10，1=敌对 10=羁绊）+ 互动模式 + 冲突点 + 演变方向。

**关系不是静态的**——标注角色之间的信息差（A 知道 B 不知道的事），这是未来戏剧冲突的燃料。

### 8. 录入台账

将全部内容写入 `tracking/characters.md`（使用 `templates/tracking-characters.md` 模版）。

## 完成标准

- [ ] 角色层级和故事功能已确认
- [ ] Wound → Lie → Want → Need 完整且因果链自洽
- [ ] 三词定义 + 致命缺陷 + 双面自我
- [ ] 弧线类型 + 5 节点 + 7 问检验通过
- [ ] 声线卡 ≥5 项 + 外在标记 ≥2 个 + 无单维度人设
- [ ] 关系网已标注（至少含与主角的关系）
- [ ] 如果是反派，动机合理 + 与主角有镜像点
- [ ] `tracking/characters.md` 已录入
- [ ] 准备进入阶段 04（如果是连锁）

## 常见错误

1. **跳过 Wound 直接写 Lie。** Lie 源自 Wound。没有 Wound 的 Lie = 读者不知道为什么他信这个。
2. **Want 和 Need 写反。** Want = 角色知道。Need = 故事教他。
3. **三词定义全同义。** "聪明+机智+敏锐" = 一个词。应该是 "聪明+社恐+嘴硬心软"。
4. **致命缺陷不会致命。** "有点急躁" = 不是致命缺陷。"为了保护信念可以牺牲一切" = 致命缺陷。
5. **所有角色同一种声线。** 遮住名字分不清 = 失败。
6. **单维度标签。** "高冷总裁" = 贴纸不是角色。叠矛盾特质。
7. **反派动机是"因为他是坏人"。** 没人是坏人为坏而坏。反派认为自己在做对的事。
8. **关系网只有友好/敌对两个温度。** 加中间地带。加信息差。
