# 网文各类型写作指南

> 精炼实用 · 覆盖八大类型 · 面向番茄/起点双平台

---

## 一、玄幻（升级流 / 废柴逆袭 / 退婚流）

### 核心爽点
- **实力碾压**：主角通过奇遇/金手指快速升级，越级打脸，以弱胜强
- **身份反转**：从被鄙视的废物到万人敬仰的强者，反差越强越爽
- **资源获取**：秘境探险、丹药法宝、天材地宝的收集与争夺

### 经典套路/模板
- **退婚流**：开局被退婚羞辱 → 激活金手指 → 三年之约打脸 → 前未婚妻后悔
- **废柴逆袭**：检测废灵根/废武魂 → 意外获得远古传承 → 一路碾压天骄
- **升级流标准节奏**：遇强敌 → 打不过 → 奇遇突破 → 反杀 → 装逼收尾

### 世界观构建要点
- 明确的等级体系（炼气→筑基→金丹→元婴，或自定义9阶/10阶），每阶有小境界
- 势力格局：宗门/家族/皇朝/散修四层结构，矛盾天然存在
- 地图扩张逻辑：新手村→郡城→王朝→大陆→星域，逐层解锁

### 常见雷区
- 等级体系混乱，战力前后矛盾
- 反派降智送人头，缺乏真正威胁
- 奇遇过多变成"寻宝清单"，失去探索感
- 女主工具人化，感情线可有可无

### 番茄/起点差异化策略
| 维度 | 番茄 | 起点 |
|------|------|------|
| 节奏 | 前3章必须出金手指+首次打脸 | 前10章铺垫世界观可接受 |
| 字数 | 800-1500字/章，节奏快 | 2000-3000字/章，描写可更细腻 |
| 爽点密度 | 章章有爽点，3章一个小高潮 | 5-10章一个阶段高潮 |
| 读者预期 | 碎片化阅读，不需要太深设定 | 愿意接受复杂设定和长线伏笔 |

## 二、都市（重生 / 系统 / 神豪 / 赘婿）

### 核心爽点
- **先知红利**：重生后利用前世记忆炒股、买房、投资，步步领先
- **降维打击**：用未来知识/技能在现代社会碾压对手
- **财富自由**：神豪系统无限金钱，花钱就是变强
- **身份逆袭**：赘婿隐藏身份（战神/医圣/财阀继承人）逐步揭露

### 经典套路/模板
- **重生流**：回到过去 → 挽救遗憾 → 报复仇人 → 建立商业帝国
- **神豪流**：绑定神豪系统 → 消费返利/花钱升级 → 买买买碾压一切
- **赘婿流**：入赘被羞辱 → 暗中调用势力 → 身份层层揭晓打脸

### 世界观构建要点
- 时间锚点要精准（哪一年、什么大事件），利用真实历史事件增加代入感
- 商业逻辑至少经得起表面推敲，不要出现"100万收购腾讯"这类硬伤
- 人物关系网：家族、商业对手、官场、军方，多线交织

### 常见雷区
- 重生后性格立刻大变，没有过渡期
- 商业操作过于儿戏（10章收购百家公司）
- 女主全是花瓶，感情线千篇一律
- 系统过于万能，失去挑战感

### 番茄/起点差异化策略
| 维度 | 番茄 | 起点 |
|------|------|------|
| 开局 | 重生即打脸，前3章必须爽 | 可以慢热铺垫人物关系 |
| 题材偏好 | 神豪、赘婿题材流量极大 | 重生商战、文娱类更受欢迎 |
| 尺度 | 暧昧擦边可，不能越线 | 相对更宽松 |

## 三、悬疑（推理 / 规则怪谈 / 刑侦 / 惊悚）

### 核心爽点
- **智力碾压**：主角通过超强推理/特殊能力破解迷局
- **规则破解**：在诡异规则中找到漏洞/生路，以智取胜
- **层层反转**：你以为的真相只是表象，真凶另有其人
- **细思极恐**：日常场景下的诡异感，越普通越恐怖

### 经典套路/模板
- **规则怪谈**：进入诡异空间 → 收到规则卡片 → 找出生路 → 规则背后有更大阴谋
- **刑侦流**：离奇案件 → 主角特殊能力（犯罪侧写/微表情/法医）→ 连环案件串并
- **本格推理**：案件发生 → 线索收集 → 多重推理 → 最终反转

### 世界观构建要点
- 规则怪谈：规则必须自洽，漏洞是设计好的逃生口而非BUG
- 刑侦：案件逻辑链完整，手法具有现实可行性的"奇观感"
- 恐怖氛围：信息差是关键——读者知道的和角色知道的要有落差

### 常见雷区
- 推理全靠"灵光一闪"，缺乏逻辑链
- 规则怪谈的规则前后矛盾
- 凶手动机牵强，为反转而反转
- 恐怖全靠跳吓（Jump Scare），缺乏心理恐怖

### 番茄/起点差异化策略
| 维度 | 番茄 | 起点 |
|------|------|------|
| 热门子类 | 规则怪谈流量最大 | 无限流、推理向更受欢迎 |
| 节奏 | 每章结尾必须有悬念钩子 | 可以多章铺垫一个案件 |
| 读者容忍度 | 轻微逻辑漏洞可接受 | 推理逻辑要求更高 |

## 四、科幻（末日 / 星际 / 赛博朋克）

### 核心爽点
- **末世生存**：物资囤积、基地建设、在废墟中重建文明
- **科技碾压**：用超越时代的技术横扫对手
- **星际征服**：舰队指挥、星球殖民、文明等级的跃升
- **人体改造**：赛博义体、基因进化、突破人类极限

### 经典套路/模板
- **末世流**：重生回末日前 → 囤物资/觉醒异能 → 建立安全区 → 对抗丧尸潮/变异兽
- **星际流**：地球进入星际时代 → 获得远古科技/战舰 → 从雇佣兵到星际霸主
- **赛博朋克**：底层改造人 → 发现公司阴谋 → 黑入系统/街头火并 → 对抗巨型公司

### 世界观构建要点
- 末世：物资清单要真实（食物保质期、燃油、武器），科技树合理
- 星际：要有明确的文明等级/科技等级体系，战力不能崩
- 赛博朋克：高科技低生活的对比要强烈，公司替代政府是核心设定

### 常见雷区
- 末世文变成"打怪升级"，失去生存压迫感
- 星际战斗写成玄幻打架（修仙机甲流）
- 赛博朋克只写科技外皮，丢了"反乌托邦"精神内核
- 科技设定前后矛盾，无视基础物理定律

### 番茄/起点差异化策略
| 维度 | 番茄 | 起点 |
|------|------|------|
| 热门题材 | 末世囤货、系统末世 | 硬科幻、星际文明 |
| 设定深度 | 不宜过深，降低阅读门槛 | 可做硬设定，读者接受度高 |
| 金手指 | 系统类最受欢迎 | 科技类/传承类均可 |

## 五、历史（架空 / 朝堂 / 种田 / 争霸）

### 核心爽点
- **知识碾压**：用现代知识（制度/科技/军事）在古代降维打击
- **种田建设**：从一村一县开始，逐步打造理想国度
- **朝堂博弈**：权谋斗争、党争站队、君臣博弈、一步步位极人臣
- **名将养成**：发掘/培养历史名将，组建无敌军团

### 经典套路/模板
- **架空争霸**：穿越成落魄皇子/小贵族 → 种田发育 → 科举/军功崛起 → 逐鹿天下
- **朝堂流**：穿越成小官 → 靠现代政治智慧站队 → 进入权力中心 → 推行改革
- **种田流**：穿越成农家子 → 改良农业/兴办工坊 → 带动一方富裕 → 影响力扩大

### 世界观构建要点
- 架空历史：朝代制度要有参考原型（唐宋明清选一个底盘），不宜四不像
- 时间线：明确标注架空，不要在真实历史时间线上制造硬伤
- 技术树：改良要循序渐进，不要直接跳到工业革命

### 常见雷区
- 现代观念强行灌输，古人秒懂秒接受
- 科举/官职体系搞错朝代
- 发明创造过于跳跃（穿越三年搞出蒸汽机）
- 战争描写过于儿戏，无视后勤和气候

### 番茄/起点差异化策略
| 维度 | 番茄 | 起点 |
|------|------|------|
| 节奏 | 快速出成果，前10章要有实质进展 | 可以慢热铺垫，细节考据 |
| 种田文 | 轻松种田+爽点结合最受欢迎 | 严谨考据种田更有市场 |
| 考据要求 | 放宽，大致合理即可 | 读者会挑历史细节硬伤 |

## 六、言情（甜宠 / 虐恋 / 追妻火葬场）

### 核心爽点
- **甜宠**：男主实力强大但只对女主温柔，无原则宠溺
- **追妻火葬场**：前期男主虐女主 → 女主离开后男主悔悟 → 疯狂追妻被打脸
- **虐恋**：深情虐心的拉扯感，爱而不得的痛苦美学
- **护短**：男主/女主护短打脸恶毒女配，正义感满足

### 经典套路/模板
- **追妻火葬场**：先婚后爱冷暴力 → 女主心死离开 → 男主发现误会追妻 → 女主不接受 → 男主持续付出终获原谅
- **甜宠文**：契约/联姻开局 → 日常互动逐渐动心 → 外部危机增进感情 → 双向奔赴HE
- **虐恋文**：身份对立（仇家/对家）→ 相爱不能相守 → 误会加深 → 生死考验 → 释怀或BE

### 世界观构建要点
- 人设是核心：男主必须有"苏点"（强大+深情+偏执中的一种组合），女主必须有成长线
- 冲突设计：外部压力（家族/身份）＞内部误会（缺乏沟通），后者容易让读者烦躁
- 甜宠文的危机感：不能只甜，要偶尔有外部威胁来凸显守护感

### 常见雷区
- 女主傻白甜无成长，全程被动
- 追妻火葬场"火葬场"不够狠，轻易原谅
- 恶毒女配脸谱化，纯粹为恶而恶
- 甜宠文甜到腻，缺乏剧情张力

### 番茄/起点差异化策略
| 维度 | 番茄 | 起点 |
|------|------|------|
| 读者画像 | 女性读者为主，年龄跨度大 | 男女读者均衡 |
| 热门类型 | 甜宠+追妻火葬场流量最大 | 古言、宅斗更受欢迎 |
| 章节节奏 | 短章高频，每章有情绪钩子 | 可以长章节精细描写感情 |

## 七、仙侠（传统仙侠 / 凡人流 / 轻松修仙）

### 核心爽点
- **修仙长生**：突破境界延长寿命，与天争命的宏大感
- **法宝神通**：获得稀有法宝、修炼绝世神通时的收获感
- **机缘奇遇**：误入秘境/古墓获得上古传承的探索感
- **轻松无敌**：开局满级/无敌流，闲庭信步碾压一切

### 经典套路/模板
- **传统仙侠**：练气筑基金丹元婴化神 → 宗门大比 → 秘境探险 → 飞升仙界
- **凡人流**：资质平庸的散修 → 靠谨慎和算计步步为营 → 资源为王 → 最终超脱
- **轻松修仙**：穿越成修仙界大佬/满级号 → 收徒装逼 → 扮猪吃虎 → 日常喜剧

### 世界观构建要点
- 修炼体系必须严密：灵气等级、功法品阶、丹药分级、法宝等级
- 势力结构：正道/魔道/散修三足鼎立，每个势力内部再分派系
- 天劫/雷劫是天然的高潮节点，要精心设计
- 轻松修仙的世界观可以更随意，但战力体系不能崩

### 常见雷区
- 修炼体系混乱，境界划分不清晰
- 法宝和功法没有稀缺感，人人都有神器
- 轻松修仙变成无脑爽，失去修仙的"道"韵
- 传统仙侠过于苦大仇深，全程压抑

### 番茄/起点差异化策略
| 维度 | 番茄 | 起点 |
|------|------|------|
| 轻松修仙 | 流量极大，首选题材 | 也有市场但竞争激烈 |
| 传统仙侠 | 节奏要快，不宜大段讲道 | 慢热讲道可接受 |
| 凡人流 | 需要降低阅读门槛 | 硬核凡人流有忠实读者 |

## 八、游戏/电竞

### 核心爽点
- **技术碾压**：主角凭借超强操作/游戏理解秀翻全场
- **战术智慧**：通过战术/套路以弱胜强，智商压制
- **职业逆袭**：从路人王/代练到职业选手/世界冠军的逆袭之路
- **装备掉落**：爆极品装备、稀有道具的收获快感

### 经典套路/模板
- **电竞职业**：路人王被发现 → 加入战队 → 内部磨合 → 联赛夺冠 → 世界赛封神
- **游戏异界**：穿越进游戏世界 → 利用游戏知识开挂 → 收服NPC → 揭开世界真相
- **重生电竞**：前世遗憾退役 → 重生巅峰期 → 弥补遗憾组建银河战舰 → 征服世界

### 世界观构建要点
- 游戏设定要有真实感：技能CD、资源机制、地图设计要经得起推敲
- 比赛体系：城市赛→省级→全国→世界赛，层级分明
- 游戏异界要保留"游戏感"：任务系统、等级、技能树、副本机制
- 职业生态写实：青训→替补→首发→明星选手，薪资和转会机制

### 常见雷区
- 游戏设定过于复杂，非玩家读者看不懂
- 对手全是菜鸟，毫无挑战性
- 电竞文只写比赛不写人物，缺乏情感线
- 游戏异界变成流水账刷怪

### 番茄/起点差异化策略
| 维度 | 番茄 | 起点 |
|------|------|------|
| 热门题材 | 游戏异界+系统流 | 电竞职业、虚拟现实 |
| 游戏设定 | 简化机制，降低理解门槛 | 可以设计复杂系统 |
| 爽点来源 | 装备掉落、升级变强 | 比赛翻盘、战术博弈 |

## 通用写作建议

1. **开篇三章定生死**：无论哪个平台，前3章必须有明确的爽点/悬念/冲突
2. **金手指要有限制**：万能金手指会摧毁故事张力，必须有代价/冷却/条件
3. **反派要有智商**：降智反派只会让主角的胜利贬值
4. **感情线要自然**：不要为了凑CP硬塞感情戏，无CP也可以很精彩
5. **断章在钩子上**：每章结尾留悬念/期待，让读者"再看一章就睡"
6. **数据是最好的反馈**：写完十章后看追读率，追读掉就调整方向
