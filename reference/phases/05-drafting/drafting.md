# 阶段 05：初稿写作

## 触发条件

用户说"写第X章""继续写""下一章"时加载本手册。

## 写前准备

**必须完成以下所有检查项后才能开始写正文。**

### 0. 检查未定稿章节

扫描 `workspace/novels/{name}/content/` 中是否存在 `chXX-draft.md` 文件。如有，告知用户："以下章节尚未定稿：[列出]。建议先定稿再开新章，避免台账数据过期。"

### 1. 确定本章目标

加载`workspace/novels/{name}/arc/`，读取剧情弧线，确认本章的核心功能（至少一项）：
- 推进主线剧情
- 展示/深化人物
- 建立/扩展世界观
- 埋设或回收伏笔
- 制造悬念/钩子

如果本章目标不明确，先与用户确认再继续。

### 2. 检查角色台账

读取 `workspace/stories/{name}/tracking/characters.md`。

确认本章涉及角色的当前状态：
- 上次登场章节和状态（受伤？疲劳？情绪？）
- 当前能力/阶位/装备
- 当前人际关系温度
- 未完成的个人弧线节点

### 3. 检查进度文档

读取 `workspace/stories/{name}/tracking/progress.md`。确认：
- 当前时间线（本章日期）
- 已埋未收的伏笔（本章是否该收？）
- 开放线索（本章是否该推进？）
- 上一章的结尾状态（本章第一拍必须接住）

### 4. 事实核验

对权威来源做事实核验：
- 世界观与设定 → grep `workspace/stories/{name}/worldbuilding.md`
- 大纲与结构 → grep `workspace/stories/{name}/outline.md`
- 项目规则与约束 → grep `workspace/stories/{name}/rules.md`

核验范围：
- 数值类：阶位等级、魔法/技能参数、距离、时间间隔、用量
- 归属类：谁说过的台词、谁拥有的物品、谁掌握的能力
- 描述类：已登场地点外观、道具特征、种族特性

**验证方法**：逐条 grep，不凭记忆推理。

### 5. 检查外部系统

读取 `workspace/stories/{name}/rules.md`，检查是否配置了外部记忆/追踪系统。如有，在写作前按 `rules.md` 中的指令查询相关信息（如语义召回当前进度、角色状态、最近事件等）。

如果 `rules.md` 不存在，按全局规则 #8 处理：告知用户并建议创建。

### 6. 检查章首格式

确认项目 `rules.md` 中定义的章首格式（日期标注等），准备好元数据。

## 正文执行

### 情况 A：从零写（章节文件为空或不存在）

1. 用 `write_file` 写入 `workspace/stories/{name}/content/chXX-draft.md`
2. 写作中遵循 `workspace/stories/{name}/rules.md` 的风格规范（如存在）
3. 写完立即进入「写后自查」

### 情况 B：已有初稿（章节文件非空，用户要求修订）

1. 先过「写后自查」全部检查项，定位问题
2. 补充「写前准备」中可能遗漏的事实核验
3. 用 `write_file` 一次性改写全章（不用多次 `patch`）
4. 改写后再过一遍「写后自查」

## 写后自查

写完正文后，立即逐项检查。**不过不交付。**

### 1. 剧情逻辑

- 本章关键事件的前提、触发、后果是否完整？
- 事件因果关系有无跳跃？（从 A 直接到 C 缺了 B）
- 同一操作出现两次结果不同时，是否已显式区分条件？

### 2. 目标一致性

- 主角本章的行为是否与当前总体目标一致？
- 如有偏离，是否是有意的（展示另一面/被迫选择）？

### 3. 情绪与关系

- 情绪变化是否有来路？（不能凭空愤怒/感动）
- 人物间关系温度是否承接前章？（不能忽冷忽热）

### 4. 身体与状态信息

- 角色的伤势/疲劳/身体限制是否被持续追踪？
- 重要道具/装备的状态是否一致？

### 5. 场景转场

- 场景切换时读者会不会迷路？（至少给出一个锚点：地点/时间/人物）
- 连续场景间是否有呼吸感？（紧张段落接松弛段落）

### 6. 章末承接

- 章末收在变化上还是画面上？（变化 > 画面 > 总结）
- 下一章第一拍能否自然接住本章结尾？

### 7. 事实一致性

- 本章所有涉及已建立事实的地方，与 `worldbuilding.md` + `outline.md` + `rules.md` 是否一致？
- 如有新增设定，是否与已有设定冲突？

### 8. 常识逻辑

- 章节内的基本因果关系是否自洽？
- 时间顺序是否合理？（不能先发生后触发）
